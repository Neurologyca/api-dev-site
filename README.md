# Neurologyca Developer Hub

Static Astro site for a curated developer resource portal.

## Commands

- `npm install` installs dependencies
- `npm run dev` starts the local Astro dev server
- `npm run build` creates the production build in `dist/`
- `npm run preview` previews the production build locally

## Content

Update links and section copy in `src/data/resources.json`.

## Assets

Brand assets live in `public/branding/`.
